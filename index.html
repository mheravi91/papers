Curated DOI landing pages for your research

Present polished summaries, metadata, and citation snippets for each publication while automatically redirecting unknown DOIs to their canonical home on doi.org.

Browse examples View on GitHub

Why it works

  • Instant DOI redirectsAny DOI path without a custom page falls back to doi.org through an edge-free static redirect.
  • SEO-ready metadataSchema.org blocks and structured content help search engines and scholarly crawlers index each paper.
  • Zero build toolingPure HTML and CSS — simply push to GitHub Pages and you are live.
  • Reusable templateDuplicate folders per DOI, update the metadata, and ship a new landing page in minutes.

Publish a new paper page

Copy any example directory, rename it to the exact DOI (including slashes), and tailor the summary, links, and BibTeX inside index.html. Commit and push — GitHub Pages handles the rest.
